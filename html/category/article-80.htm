vscode如何运行JavaWeb项目

2、使用vscode打开(或者手动创建)Java web项目

3.点击选中该项目文件夹后点击最上面的查看--》选中命令面板后利用命令行tomcat:Generate War Package from Current Folder将项目打包成war，在项目文件下选择第一个命令会生成该项目文件夹命名的一个war包

4.添加Tomcat服务，在侧边栏 Tomcat Server 点击添加，选择自己的Tomcat目录

5.运行服务和项目，右键生成的war包，选择 Run on Tomcat Server

6.查看项目，直接在浏览器中输入http://localhost:8080或者右键发布的项目open in Browser，查看项目运行结果。
