python代码模板

作用：
代码模版可以让我们IDE创建的每一个文件 会有一个初始化的格式 下面是对于py 文件的常用模板
##Pycharm设置
File-Settings-Editor- Code Style - File and Code Templates - Python Script 中

#!/usr/bin/env python # -*- coding: utf-8 -*- """ PROJECT_NAME: ${PROJECT_NAME} ;  FILE_NAME: ${NAME} ; AUTHOR: reno ; WORK_NUMBER: your_num E_MAIL: renoyuan@foxmail.com DATE: ${DATE} ; PRODUCT_NAME: ${PRODUCT_NAME} ; """
