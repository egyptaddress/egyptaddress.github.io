Python - venv 管理虚拟环境

1. 创建一个虚拟环境

windows系统， 使用cmd窗口

# 切换到指定的目录 E:   // 切换到E盘根目录dir  // 查看文件夹中的文件cd myvenvs // 切换到指定目录 python -m venv test_venv(虚拟环境名)

2.启用虚拟环境

# 进入到创建的虚拟环境文件夹cd venv_testcd Scripts# 进入到Scripts文件夹中 activate.bat# 执行bat文件, 激活虚拟环境# 启用、激活虚拟环境后，提示符钱满会出现虚拟环境的名字
