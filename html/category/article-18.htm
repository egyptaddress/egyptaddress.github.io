Java程序查找数字的偶数因数之和，代码如下-

示例

import java.util.*; import java.lang.*; public class Demo{    public static int factor_sum(int num){       if (num % 2 != 0)       return 0;       int result = 1;       for (int i = 2; i = 2)          result *= (1 + num);       return result;    }    public static void main(String argc[]){       int num = 36;       System.out.println("该数的偶数因数之和为 ");       System.out.println(factor_sum(num));    } }

输出结果

该数的偶数因数之和为 78

一个名为Demo的类包含一个名为factor_sum的函数。这个函数查找一个数的因子，并将偶数因子相加，然后返回这个值作为输出。在main函数中，定义了需要找到偶因数的数，并对该数调用函数。相关的消息显示在控制台上。
