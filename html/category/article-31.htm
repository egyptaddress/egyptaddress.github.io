Git回退代码到指定版本方法

  • 首页
  • 文章分类
  • 正文

1. 查看所有的历史版本，获取你git的某个历史版本的id， git log
2. 回退本地代码库：git reset --hard ID
3. 推送到远程服务器：git push -f -u origin master
4. 重新拉代码：git pull
