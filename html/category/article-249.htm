element树形数据与懒加载失效

  • 首页
  • 文章分类
  • 正文

这个bug 卡了我好久 我以为是我的代码有问题 但是 发现我的版本没有这个功能

table树形数据设置，不显示下拉箭头

将官网相关demo复制到本地仍不能正常工作，没有报错

后面百度了好久才发现

版本 v2.7.0 开始增加了对 table 树形结构的支持直接让我emo了

之前我的版本是2.4.1  现在跟新之后就好了

解决方法是：

1-删除之前的版本 node.js ：  npm uninstall element-ui

2-更新指定的版本 node.js ：  npm install element-ui@2.7.0 -S
